le bit “chiffre binaire”

De l’anglais Bit ” binary digit” ( Fr, chiffre binaire) : le bit est la plus petite unité que traite un ordinateur. Groupé par huit (0110101), ils forment un octet, espace mémoire nécessaire pour inscrire un caractère préalablement défini et unité supérieure au bit de l'écriture informatique. Cette transformation se fait en correspondance avec le code ascii ( A => 1000001, B => 1000010, ...)
